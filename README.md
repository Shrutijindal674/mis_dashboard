# IITMIS Dashboard (Local Host)

This is a Vite + React + Tailwind project containing the single-file dashboard (src/App.jsx).

## What is included
- Full source code
- `package.json` with dependencies (React, Recharts, Tailwind, Vite)

## What is NOT included
- `node_modules/` (pre-downloaded packages). Shipping node_modules is usually very large and OS-specific.
  You will install them locally with `npm install` (or `pnpm i` / `yarn`).

## Requirements
- Node.js 18+ recommended

## Run locally
```bash
npm install
npm run dev
```

Then open the URL printed in the terminal (usually http://localhost:5173).

## Production build
```bash
npm run build
npm run preview
```
